Welcome to **Katari**. このページは仮の placeholder です。あとで本物の内容に置き換えてください。

Katari は agent オーケストレーションのための言語です。 `.ktr` ソースをコンパイルして
JSON IR を生成し、ランタイムで実行します。

## What you can do here

- 構文と型システムを学ぶ
- ツールチェイン (compiler / runtime / lsp) の使い方を確認する
- サンプルコードを動かす
